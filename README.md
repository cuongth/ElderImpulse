# ElderImpulse
Impulse System for the price bars

* Bullish price bar:
    (13-period EMA > previous 13-period EMA) and 
    (MACD-Histogram > previous period's MACD-Histogram)
* Bearish price bar:
    (13-period EMA < previous 13-period EMA) and 
    (MACD-Histogram < previous period's MACD-Histogram)
* Neutral price bar:
    when conditions for a bullish price bar or bearish price bar are not met.
